Тестовый проект 

Задача: 
- реализовать древовидную структуру данных, узлы которой могут содержать данные не менее трёх типов на выбор (выбраны int, double, std::string), а также могут иметь неограниченное количество дочерних узлов;
- реализовать фукции сериализации в текстовый файл и десериализации из текстового файла, формат файла не регламентирован;
- использовать только библиотеку STL.


Выполнено:
- реализована требуемая структура данных;
- наполнение структуры реализовано методом добавления множества дочерних узлов к узлу (оператор [] выполняет поиск  по значению узла, к которому будут добавлены дочерние);
- реализована сериализация структуры в файл .txt и десериализация структуры из файла;
- структура может содержать текст с пробелами и переходом на новую строку (\n), а также повторяющиеся значения;
- словарь специальных фраз-делимитеров содержится в файле node.h;
- проверка выполняется путем сериализации созданной структуры в файл file1.txt, десериализации нового экземпляра структуры из этого файла, сериализации новой структуры в файл file2.txt и посимвольным сравнением полученных файлов.
